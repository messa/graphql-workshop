GraphQL workshop
================

⚠️ WORK IN PROGRESS

Github: https://github.com/messa/pycon-graphql-workshop

This guide: https://pycon-graphql.now.sh/


Goals of this workshop
----------------------

- learn a bit about aiohttp & asyncio
- learn how to make basic React website using Next.js
- learn about GraphQL

  - server-side use in Python
  - client-side in Javascript

    - library Relay Modern


This workshop is designed that you follow this guide on your own, at our own pace.

Ask the mentors any question any time.
If you get stuck, ask for help please.
If you don't understand something, start a discussion about it.


Our project: "Polls"
--------------------

This repository contains base structure of the project you will be working on.
It's a "simple" poll application - you can list all polls, add a poll, and vote in a poll.
